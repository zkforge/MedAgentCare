中国2型糖尿病防治指南（2024年版）

发布机构：中华医学会内分泌学分会
发布年份：2024年
指南类型：临床诊疗指南
证据等级：A级

一、诊断标准

1. 糖尿病诊断标准（满足任一条即可诊断）

空腹血糖（FPG）：
- ≥7.0 mmol/L (126 mg/dL)
- 空腹定义：至少8小时未进食

或

餐后2小时血糖（2hPG）：
- ≥11.1 mmol/L (200 mg/dL)
- OGTT试验（75g葡萄糖负荷）后2小时

或

随机血糖：
- ≥11.1 mmol/L + 典型症状（多饮、多尿、多食、体重下降）

或

糖化血红蛋白（HbA1c）：
- ≥6.5%（中国2024年新增，与国际接轨）
- 需使用标准化检测方法

2. 糖尿病前期（糖调节受损）

空腹血糖受损（IFG）：
- 6.1 ≤ FPG < 7.0 mmol/L

或

糖耐量受损（IGT）：
- 7.8 ≤ 2hPG < 11.1 mmol/L

或

HbA1c异常（新增）：
- 5.7% ≤ HbA1c < 6.5%

**意义**：糖尿病前期每年约5-10%进展为糖尿病，需要生活方式干预。

二、治疗目标

1. 血糖控制目标

一般成年患者：
- HbA1c < 7.0%（首要目标）
- 空腹血糖 4.4-7.0 mmol/L
- 餐后2小时血糖 <10.0 mmol/L

更严格目标（HbA1c < 6.5%）适用于：
- 新诊断、病程短（<5年）
- 预期寿命长（>15年）
- 无心血管并发症
- 低血糖风险低

放宽目标（HbA1c < 8.0%）适用于：
- 老年人（>75岁）、预期寿命<5年
- 有严重低血糖史
- 微血管并发症晚期
- 合并严重疾病

2. 综合管理目标

血压：
- <130/80 mmHg（糖尿病合并高血压）
- 老年人（≥65岁）：<140/90 mmHg

血脂：
- LDL-C <2.6 mmol/L（一般患者）
- LDL-C <1.8 mmol/L（合并心血管疾病）
- 甘油三酯 <1.7 mmol/L

体重：
- BMI 18.5-23.9（中国标准）
- 超重/肥胖者：减重5-10%

三、生活方式干预

1. 医学营养治疗（MNT）

总能量：
- 超重/肥胖：每日减少500-750kcal
- 正常体重：维持能量平衡

碳水化合物：
- 占总能量45-60%
- 优先选择低GI食物（全谷物、豆类、蔬菜）
- 限制精制糖、含糖饮料

蛋白质：
- 占总能量15-20%
- 优质蛋白（鱼、禽、蛋、奶、大豆）
- 肾功能正常：1.0-1.2 g/kg/天
- 肾功能不全：0.8 g/kg/天

脂肪：
- 占总能量20-35%
- 限制饱和脂肪（<7%总能量）
- 增加不饱和脂肪（橄榄油、坚果、鱼油）

膳食纤维：
- 每日≥25g（蔬菜、水果、全谷物）

2. 运动锻炼

有氧运动：
- 频率：每周至少150分钟中等强度有氧运动（或75分钟高强度）
- 类型：快走、慢跑、游泳、骑车
- 建议：每周至少5天，每次30-60分钟

抗阻运动（力量训练）：
- 频率：每周2-3次
- 类型：举重、弹力带、俯卧撑、深蹲
- 建议：锻炼主要肌肉群

注意事项：
- 运动前后监测血糖，预防低血糖
- 使用胰岛素或磺脲类药物者，运动前补充碳水化合物
- 有增殖性视网膜病变、严重自主神经病变者，避免剧烈运动

四、药物治疗（最新进展）

1. 口服降糖药

二甲双胍（首选一线药物）：
- 适应证：所有2型糖尿病患者（除非禁忌）
- 优势：降糖效果好、不增加体重、心血管获益、价格低
- 剂量：从500mg qd或bid开始，逐渐加量至2000-2550mg/天
- 禁忌：eGFR<30 ml/min/1.73m²、肝功能不全、酗酒、乳酸酸中毒史
- 副作用：胃肠道反应（恶心、腹泻），缓释剂型耐受性更好

【重点】SGLT2抑制剂（钠-葡萄糖协同转运蛋白2抑制剂）- 新型降糖药：
- 代表药物：达格列净、恩格列净、卡格列净
- 机制：抑制肾脏近端小管葡萄糖重吸收，促进尿糖排出
- 降糖效果：HbA1c降低0.5-1.0%
- **心血管获益**（重大突破）：
  * 降低心血管死亡风险25-38%（EMPA-REG、DECLARE-TIMI 58研究）
  * 降低心衰住院风险30-35%
  * 适用于合并心血管疾病、心衰、慢性肾病的糖尿病患者
- **肾脏保护**：
  * 延缓肾功能下降（eGFR下降速度减慢50%）
  * 减少白蛋白尿
  * 适用于CKD 1-4期（eGFR≥20 ml/min）
- 额外获益：
  * 减轻体重（2-3kg）
  * 降低血压（收缩压降低3-5 mmHg）
  * 降低尿酸
- 副作用：
  * 泌尿生殖道感染（10%，女性更常见）
  * 低血容量（脱水、头晕、低血压）
  * 酮症酸中毒（罕见，<0.1%，多见于低碳水化合物饮食、禁食、急性疾病）
- 禁忌：eGFR<20 ml/min、1型糖尿病（酮症酸中毒风险）
- **最新研究**（2023-2024）：
  * DAPA-CKD研究：达格列净降低CKD进展风险39%
  * EMPEROR-Preserved研究：恩格列净改善射血分数保留性心衰预后

【重点】GLP-1受体激动剂（胰高糖素样肽-1受体激动剂）- 新型降糖药：
- 代表药物：利拉鲁肽、度拉糖肽、司美格鲁肽、替尔泊肽
- 机制：
  * 促进葡萄糖依赖性胰岛素分泌
  * 抑制胰高糖素分泌
  * 延缓胃排空
  * 增强饱腹感、减少食欲
- 降糖效果：HbA1c降低0.8-1.8%（司美格鲁肽、替尔泊肽效果最强）
- **心血管获益**（重大突破）：
  * 降低主要心血管事件（心梗、脑卒中、心血管死亡）风险12-26%
  * LEADER研究（利拉鲁肽）：降低心血管死亡22%
  * SUSTAIN-6研究（司美格鲁肽）：降低脑卒中风险39%
  * REWIND研究（度拉糖肽）：降低心血管事件12%
- **减重效果**（显著）：
  * 利拉鲁肽：减重2-3kg
  * 司美格鲁肽：减重5-15kg（最强减重效果）
  * 替尔泊肽（GLP-1/GIP双激动剂）：减重10-20kg（超越司美格鲁肽）
- 给药方式：皮下注射（每日1次或每周1次）
  * 每日注射：利拉鲁肽
  * 每周注射：度拉糖肽、司美格鲁肽、替尔泊肽（依从性更好）
- 副作用：
  * 胃肠道反应（恶心、呕吐、腹泻，30-40%）
  * 低血糖风险低（单独使用时）
  * 急性胰腺炎（罕见，<0.1%）
  * 甲状腺C细胞肿瘤（动物实验，人类无明确证据）
- 禁忌：甲状腺髓样癌家族史、2型多发性内分泌腺瘤综合征
- **最新研究**（2023-2024）：
  * SELECT研究：司美格鲁肽降低超重/肥胖非糖尿病人群心血管事件20%
  * SURMOUNT研究：替尔泊肽（GLP-1/GIP双激动剂）减重效果超过司美格鲁肽，HbA1c降低2.0-2.3%
  * 口服GLP-1（口服司美格鲁肽）已上市，依从性更佳

DPP-4抑制剂（二肽基肽酶-4抑制剂）：
- 代表药物：西格列汀、沙格列汀、利格列汀
- 优势：低血糖风险低、不增加体重、耐受性好
- 降糖效果：HbA1c降低0.5-0.8%（弱于SGLT2i和GLP-1RA）
- 适用：不耐受二甲双胍、不愿注射GLP-1RA者

磺脲类（传统降糖药）：
- 代表药物：格列美脲、格列吡嗪、格列齐特
- 优势：降糖效果好、价格低
- 劣势：低血糖风险高、体重增加
- 适用：经济困难、新药不可及者

α-葡萄糖苷酶抑制剂：
- 代表药物：阿卡波糖、伏格列波糖
- 机制：延缓碳水化合物吸收，降低餐后血糖
- 适用：餐后高血糖为主、中国饮食（碳水化合物为主）
- 副作用：胃肠胀气、腹泻（从小剂量开始，逐渐加量）

噻唑烷二酮类：
- 代表药物：吡格列酮
- 优势：改善胰岛素抵抗、降低心血管风险
- 劣势：体重增加、水肿、骨折风险↑、膀胱癌风险↑（有争议）
- 适用：胰岛素抵抗明显者

2. 注射降糖药

胰岛素：
- 适应证：
  * 1型糖尿病（必须使用）
  * 2型糖尿病口服药控制不佳（HbA1c>9.0%或空腹血糖>11.1 mmol/L）
  * 妊娠糖尿病
  * 急性并发症（酮症酸中毒、高渗性昏迷）
- 类型：
  * 速效胰岛素：门冬胰岛素、赖脯胰岛素（餐时注射）
  * 长效胰岛素：甘精胰岛素、德谷胰岛素（每日1次基础胰岛素）
  * 预混胰岛素：含速效+中效成分（每日2次）
- 起始方案：
  * 基础胰岛素：0.1-0.2 U/kg/天，睡前注射
  * 根据空腹血糖调整剂量（每2-3天调整2U）
  * 目标：空腹血糖4.4-7.0 mmol/L

五、联合用药策略（最新推荐）

1. 初始治疗（新诊断糖尿病）

HbA1c < 9.0%：
- 二甲双胍单药
- 若3个月未达标，加用第二种药物

HbA1c 9.0-10.0%：
- 二甲双胍 + SGLT2i 或 GLP-1RA（优先选择，心肾获益）
- 或 二甲双胍 + DPP-4i（经济考虑）

HbA1c > 10.0% 或症状明显：
- 直接起始胰岛素治疗
- 血糖控制后可转为口服药

2. 二线药物选择（二甲双胍基础上）

合并心血管疾病或心血管高危：
- **首选 SGLT2i 或 GLP-1RA**（心血管获益明确）
- 不推荐：磺脲类、噻唑烷二酮类

合并心力衰竭：
- **首选 SGLT2i**（降低心衰住院风险）

合并慢性肾病（eGFR 30-60）：
- **首选 SGLT2i 或 GLP-1RA**（肾脏保护）
- 避免：二甲双胍（eGFR<30时禁用）

合并肥胖（BMI≥28）：
- **首选 GLP-1RA 或 SGLT2i**（减重效果）
- 避免：磺脲类、胰岛素（增加体重）

经济困难、药物不可及：
- 磺脲类（格列美脲、格列吡嗪）
- α-葡萄糖苷酶抑制剂（阿卡波糖）

3. 三联治疗

若二甲双胍 + 第二种药物仍未达标：
- 二甲双胍 + SGLT2i + GLP-1RA（推荐组合，心肾获益叠加）
- 二甲双胍 + SGLT2i + DPP-4i
- 二甲双胍 + GLP-1RA + 基础胰岛素

注意：SGLT2i + GLP-1RA 联合使用安全性好，协同降糖、减重、心肾保护。

六、并发症筛查与管理

微血管并发症：
- 糖尿病肾病：每年尿白蛋白/肌酐比值（UACR）、血肌酐、eGFR
- 糖尿病视网膜病变：每年眼底检查（散瞳眼底照相或OCT）
- 糖尿病神经病变：每年足部检查（10g尼龙丝、音叉振动觉）

大血管并发症：
- 心血管疾病：心电图（每年）、超声心动图（必要时）、冠脉CT或造影（高危）
- 脑血管疾病：颈动脉超声（每年）
- 下肢血管病变：足背动脉搏动、踝肱指数（ABI）

七、糖尿病教育与自我管理

血糖监测：
- 口服药治疗：每周1-2次空腹+餐后2小时血糖
- 胰岛素治疗：每日多次（空腹、三餐前、餐后2小时、睡前）
- 每3个月复查HbA1c

低血糖预防与处理：
- 识别低血糖症状：心慌、出汗、饥饿感、头晕、手抖
- 轻度低血糖（3.0-3.9 mmol/L）：口服15g碳水化合物（3-4块糖、半杯果汁）
- 重度低血糖（<3.0 mmol/L或意识障碍）：静脉注射葡萄糖或胰高糖素注射

足部护理：
- 每日检查足部（水疱、破损、红肿）
- 保持足部清洁干燥
- 穿合适的鞋袜
- 避免赤脚行走
- 定期修剪趾甲（避免过短或尖锐）
