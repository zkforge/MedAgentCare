中国高血压防治指南（2023年修订版）- 完整版

发布机构：中华医学会心血管病学分会
发布年份：2023年
指南类型：临床诊疗指南
指南版本：最新修订版
证据等级：A级（基于多项大型随机对照试验）

【核心更新要点】
- 诊断标准维持不变：≥140/90mmHg（与WHO、ESC标准一致）
- 强调家庭血压监测的重要性
- 细化特殊人群（老年人、糖尿病、肾病）的治疗目标
- 推荐单片复方制剂提高依从性
- 新增数字化管理建议

一、诊断标准（核心内容）

【基本诊断标准】
所有成年人（≥18岁）统一标准：
- 诊室血压：收缩压≥140mmHg 和/或 舒张压≥90mmHg
- 测量要求：非同日3次测量
- 排除：白大衣高血压、继发性高血压

【重要说明】
- 美国2017年指南将标准降至≥130/80mmHg，但中国未跟进
- 中国保持≥140/90mmHg标准，与欧洲、WHO一致
- 理由：中国人群脑卒中风险更高，过早治疗利大于弊

1. 血压测量要求

诊室血压测量（金标准）：
- 环境：安静房间，测量前休息5-10分钟
- 体位：坐位，背部支撑，双脚平放，上臂与心脏同高
- 设备：使用校准的电子血压计或水银柱血压计
- 操作：测量2-3次，每次间隔1-2分钟，取平均值
- 频率：非同日3次测量，若≥140/90mmHg即可诊断

家庭血压监测（推荐）：
- 标准：≥135/85mmHg（注意：低于诊室标准5mmHg）
- 时间：早晨（起床1小时内）和晚上（睡前）
- 频率：每次测2-3遍，取平均值
- 天数：连续测量7天，去掉第1天数据，取后6天平均值
- 优势：更接近真实血压，排除白大衣高血压

24小时动态血压监测（ABPM）：
- 标准：24小时平均≥130/80mmHg，白天≥135/85mmHg，夜间≥120/70mmHg
- 适应证：疑似白大衣高血压、难治性高血压、血压波动大
- 优势：了解昼夜节律（杓型、非杓型、反杓型）

2. 高血压分级（所有成年人统一）

正常血压：
- 正常：收缩压<120mmHg 且 舒张压<80mmHg
- 正常高值：120-139/80-89mmHg（高危人群需预防）

高血压分级：
- 1级高血压（轻度）：140-159/90-99mmHg
- 2级高血压（中度）：160-179/100-109mmHg
- 3级高血压（重度）：≥180/110mmHg
- 单纯收缩期高血压：收缩压≥140mmHg 且 舒张压<90mmHg（老年人常见）

【特殊说明】
- 无论年龄、性别、种族，诊断标准相同
- 老年人、糖尿病患者诊断标准不降低（治疗目标有差异，见后）
- 青少年高血压参照年龄、性别、身高的百分位标准（<18岁）

3. 诊断流程

初步筛查：
- 所有成年人每年至少测量1次血压
- 高危人群（肥胖、家族史、糖尿病）每3-6个月测1次

诊断流程：
步骤1：诊室血压≥140/90mmHg → 怀疑高血压
步骤2：非同日重复测量2次（共3次），若2次以上≥140/90mmHg → 确诊高血压
步骤3：首次诊断需完善以下检查：
   - 排查继发性高血压：肾功能、肾上腺CT/MRI、肾动脉超声
   - 评估靶器官损害：心电图、超声心动图、眼底、尿常规、肾功能
   - 评估心血管风险：血脂、血糖、尿酸、颈动脉超声

特殊情况：
- 白大衣高血压：诊室血压≥140/90mmHg，但家庭血压<135/85mmHg（暂不用药，密切随访）
- 隐匿性高血压：诊室血压<140/90mmHg，但家庭血压≥135/85mmHg（需治疗，预后差）
- 高血压急症：血压≥180/120mmHg 伴急性靶器官损害（需立即降压）

4. 特殊人群诊断标准差异

【重要】虽然诊断标准统一，但以下人群需特别注意：

老年人（≥65岁）：
- 诊断标准：与青壮年相同（≥140/90mmHg）
- 特点：
  * 单纯收缩期高血压常见（血管硬化所致）
  * 血压波动大，易发生体位性低血压
  * 白大衣高血压比例更高
- 测量注意事项：
  * 需测量立位血压（排除体位性低血压）
  * 优先推荐家庭血压监测或ABPM
  * 避免过度降压导致灌注不足

合并糖尿病患者：
- 诊断标准：与一般人群相同（≥140/90mmHg）
- 特点：
  * 高血压患病率更高（约60%）
  * 心血管风险显著升高
  * 易合并肾脏损害、视网膜病变
- 筛查频率：每3个月测1次血压
- 治疗目标：更严格（<130/80mmHg）

合并慢性肾脏病（CKD）患者：
- 诊断标准：相同（≥140/90mmHg）
- 特点：
  * 高血压加速肾功能恶化
  * 易发生难治性高血压
  * 需监测血钾（ACEI/ARB可致高钾血症）
- 治疗目标：<130/80mmHg（减缓肾病进展）

妊娠期高血压：
- 诊断标准：≥140/90mmHg（妊娠20周后新发）
- 分类：
  * 妊娠期高血压：单纯血压升高
  * 子痫前期：血压升高 + 蛋白尿
  * 子痫：子痫前期 + 抽搐
- 治疗目标：140-150/90-100mmHg（过度降压影响胎盘灌注）

儿童和青少年（<18岁）：
- 诊断标准：不同于成人，参照年龄、性别、身高百分位表
- 高血压定义：血压≥同年龄、性别、身高第95百分位
- 需排除继发性高血压（儿童高血压50%为继发性）

二、治疗目标（按人群分层，精准控制）

【总体原则】
- 个体化治疗：根据年龄、合并症、耐受性调整目标
- 分阶段达标：初期目标<150/90mmHg，维持期目标<140/90mmHg，理想目标<130/80mmHg
- 避免过度降压：舒张压不低于60mmHg（老年人不低于65mmHg）
- 获益优先：优先控制收缩压（脑卒中风险更相关）

1. 一般成年人（18-64岁，无合并症）

初始目标（启动治疗后1-3个月）：
- <150/90mmHg（快速降低风险）

维持目标（3个月后）：
- <140/90mmHg（标准目标，所有患者应达到）

理想目标（能耐受且无不良反应）：
- <130/80mmHg（进一步降低心血管风险，但需个体化评估）

注意事项：
- 降压速度：2-4周内逐渐达标，避免过快降压
- 监测症状：头晕、乏力、视力模糊提示降压过度
- 评估耐受性：若无不良反应，可尝试<130/80mmHg

2. 老年人（≥65岁）

【核心原则】获益与安全平衡，避免过度降压导致跌倒、肾功能下降

65-79岁（年轻老年人）：
- 初始目标：<150/90mmHg
- 维持目标：<140/90mmHg（耐受性好的患者）
- 理想目标：<130/80mmHg（身体状况良好、无体位性低血压者）
- 降压速度：比年轻人更缓慢，观察4-8周

≥80岁（高龄老年人）：
- 目标：<150/90mmHg（不推荐更严格控制）
- 原因：过度降压增加跌倒、骨折、肾损伤风险
- 特殊考虑：
  * 若已达<140/90mmHg且耐受良好，可维持
  * 衰弱、多重用药、认知障碍者，目标可放宽至<160/90mmHg

老年人特殊监测：
- 立位血压：每次随访测量，排除体位性低血压（立位血压下降≥20/10mmHg）
- 肾功能：每3-6个月监测，避免过度降压导致肾灌注不足
- 认知功能：关注头晕、嗜睡、记忆力下降
- 日常活动能力：避免降压影响生活质量

3. 合并糖尿病患者

目标血压：<130/80mmHg（严格控制，减少微血管并发症）

【核心理由】
- 糖尿病+高血压→心血管风险增加2-4倍
- 严格控制血压可显著减少：
  * 糖尿病肾病进展（50%）
  * 糖尿病视网膜病变（33%)
  * 心血管事件（20-25%）
  * 脑卒中（40%）

分层目标：
- 糖尿病肾病（尿白蛋白/肌酐>30mg/g）：<130/80mmHg，甚至<120/75mmHg
- 单纯糖尿病（无肾病）：<130/80mmHg
- 合并冠心病：<130/80mmHg，但舒张压≥70mmHg（避免冠脉灌注不足）

监测要点：
- 血糖：部分降压药影响血糖代谢（β受体阻滞剂、噻嗪类利尿剂）
- 血钾：ACEI/ARB联合使用需警惕高钾血症
- 肾功能：每3个月监测血肌酐、尿白蛋白
- 眼底：每年1次眼底检查

4. 合并冠心病患者

目标血压：<130/80mmHg，但舒张压≥70mmHg

【重要警告】舒张压过低（<60mmHg）反而增加心肌梗死风险（冠脉灌注压↓）

分层目标：
- 稳定型冠心病：<130/80mmHg，舒张压≥70mmHg
- 急性冠脉综合征（ACS）后：<130/80mmHg，但初期避免过度降压
- 心肌梗死后：<130/80mmHg，优先使用β受体阻滞剂+ACEI

监测要点：
- 心绞痛症状：降压后胸痛加重提示灌注不足
- 心电图：定期监测心肌缺血征象
- 心脏标志物：定期复查肌钙蛋白、BNP
- 心率：β受体阻滞剂使用后，目标心率55-60次/分

5. 合并慢性肾脏病（CKD）患者

目标血压：<130/80mmHg（延缓肾功能恶化）

【核心目标】减少蛋白尿、保护肾功能

分层目标：
- CKD 1-3期（eGFR≥30ml/min/1.73m²）：<130/80mmHg
- CKD 4-5期（eGFR<30ml/min/1.73m²）：<140/90mmHg（避免过度降压）
- 大量蛋白尿（>1g/天）：<125/75mmHg（更严格控制）

药物选择（见第四部分）：
- 首选：ACEI 或 ARB（减少蛋白尿、保护肾功能）
- 禁忌：双侧肾动脉狭窄患者禁用ACEI/ARB
- 监测：血钾、血肌酐（用药后1-2周复查）

监测要点：
- 肾功能：每3个月监测血肌酐、eGFR
- 尿蛋白：每3个月监测尿白蛋白/肌酐比值
- 血钾：ACEI/ARB可致高钾血症（血钾>5.5mmol/L需调整）
- 贫血：CKD患者易合并贫血，影响血压控制

6. 合并脑血管病患者

急性期（发病2周内）：
- 脑梗死：血压<220/120mmHg不干预（维持脑灌注）
- 脑出血：收缩压<140mmHg（减少再出血）
- 溶栓治疗：血压<185/110mmHg

恢复期（发病2周后）：
- 目标血压：<140/90mmHg
- 能耐受者：<130/80mmHg
- 降压速度：缓慢，避免脑灌注不足

一级预防（无脑血管病史）：
- 目标血压：<130/80mmHg（显著减少首次脑卒中风险）

7. 合并心力衰竭患者

射血分数降低性心衰（HFrEF，EF<40%）：
- 目标血压：<130/80mmHg
- 药物首选：ACEI/ARB + β受体阻滞剂 + 利尿剂（"金三角"）
- 注意：初始治疗血压可能降低，需逐渐加量

射血分数保留性心衰（HFpEF，EF≥50%）：
- 目标血压：<130/80mmHg
- 药物首选：ACEI/ARB + 利尿剂
- 注意：严格控制血压可改善症状

8. 妊娠期高血压

轻中度妊娠期高血压：
- 目标血压：140-150/90-100mmHg
- 原因：过度降压影响胎盘灌注、胎儿发育

重度妊娠期高血压或子痫前期：
- 目标血压：<160/110mmHg（预防脑出血、抽搐）
- 紧急情况：静脉降压药（硫酸镁、拉贝洛尔）

产后：
- 多数妊娠期高血压产后6周内自行恢复
- 若持续升高，按一般高血压治疗

9. 难治性高血压

定义：使用≥3种降压药（包括利尿剂）足量治疗，血压仍≥140/90mmHg

治疗目标：
- <140/90mmHg（可能需要4-5种药物）

策略：
- 排查继发性高血压（肾动脉狭窄、原发性醛固酮增多症）
- 评估依从性（药物是否规律服用）
- 加用螺内酯（醛固酮受体拮抗剂）
- 考虑肾交感神经去除术（难治性高血压的介入治疗）

10. 特殊职业人群

飞行员、驾驶员、高空作业者：
- 目标血压：<140/90mmHg
- 注意：避免使用可能影响注意力的药物（α受体阻滞剂）
- 监测：定期体检，确保血压稳定

运动员：
- 目标血压：<140/90mmHg
- 药物选择：避免利尿剂（兴奋剂检测）、β受体阻滞剂（影响运动能力）

三、生活方式干预

1. 饮食干预
   - 低盐饮食：每日钠摄入<2g（食盐<5g）
   - DASH饮食：富含蔬菜、水果、全谷物、低脂奶制品
   - 限制饮酒：男性<25g/天，女性<15g/天

2. 运动锻炼
   - 中等强度有氧运动：每周150分钟
   - 每周3-5次，每次30-60分钟
   - 推荐运动：快走、游泳、骑车

3. 控制体重
   - 目标BMI：18.5-24
   - 腰围：男性<90cm，女性<85cm

4. 其他措施
   - 戒烟
   - 减轻精神压力
   - 规律作息

四、药物治疗

1. 起始治疗
   - 1级高血压低危：生活方式干预3个月，无效后启动药物
   - 1级高血压中高危：立即启动药物治疗
   - 2级及以上高血压：立即启动药物治疗

2. 一线降压药物
   - 钙通道阻滞剂（CCB）：如硝苯地平、氨氯地平
   - ACEI：如依那普利、卡托普利
   - ARB：如缬沙坦、替米沙坦
   - 利尿剂：如氢氯噻嗪、吲达帕胺
   - β受体阻滞剂：如美托洛尔、比索洛尔

3. 联合用药
   - 单药控制不佳：联合2种或以上药物
   - 推荐组合：ACEI/ARB + CCB
   - 推荐组合：ACEI/ARB + 利尿剂
   - 推荐组合：CCB + 利尿剂

4. 特殊人群
   - 糖尿病：首选ACEI/ARB
   - 冠心病：首选β受体阻滞剂 + ACEI/ARB
   - 心衰：首选ACEI/ARB + β受体阻滞剂 + 利尿剂
   - 肾病：首选ACEI/ARB

五、随访管理

1. 随访频率
   - 新诊断或调整用药：2-4周复诊
   - 血压控制稳定：1-3个月复诊
   - 高危患者：加强随访

2. 监测指标
   - 血压：家庭血压监测（每日早晚各1次）
   - 心率
   - 靶器官损害：心、脑、肾、眼底、血管
   - 代谢指标：血脂、血糖、尿酸

3. 患者教育
   - 高血压知识普及
   - 用药依从性教育
   - 自我血压监测培训
   - 生活方式指导
